There are different programming languages that involve web development

HTML (Hypertext Markup Language)

It is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content.

CSS (Cascading Style Sheets)

Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML.

JavaScript

JavaScript, often abbreviated as JS, is a programming language and core technology of the Web, alongside HTML and CSS. 99% of websites use JavaScript on the client side for webpage behavior.
